Automatic Colony Counter LZ-ACC-A112

Automatic Colony Counter LZ-ACC-A112 is equipped with high resolution Camera lens and low distortion optical lens. Incorporated with advanced Smart Counter software for accurate automatic microbial colony counting. Open platform design, bottom LED dual light source makes it user friendly counter. Encrypted USB-key guarantees data security, and convenient data management.

Specifications

Camera True Color high definition camera
Camera Resolution 8.29 millions pixels
Counting Software Smart counter software
Colony Size Resolution > 0.1 mm
Colony Color Recognition 8 colors
Applicable Methods Plate Type Spread plate, Pour plate, Membrane filter, Spiral plate
Counting Speed ≤ 5 secs
Light Source Long life dual LED light
Working Temperature 0 to 35°C
Material ABS plastic
Size of Petri dish < 120 mm
Electrical Supply AC 110 to 220 V, 50 Hz
Host Dimension 290 × 260 × 380 mm
Weight 3 kg

Features

  • 360 degrees LED dual light source
  • High resolution Camera with low distortion optical lens
  • Smart counter operational interface
  • No need to install any drivers or software, software is directly copied and used
  • Automatic data processing and storage
  • Quick counting of the number of colonies
  • Features 3 levels of Authority Management
  • Adhering colonies are automatically segmented by smart counter Intelligent colony identification technology
  • Colonies on the grid filters can be easily identified
  • Comes with features like color adaption settings, colony ratio and diameter settings for the improved counting accuracy
  • Distinguishes 8 different colored colonies
  • Manual addition and deletion of colonies
  • Can set unwanted (non-countable) area

Applications

Used in automatic colony counting, reading for quantitative studies, quality and research purpose across microbiology, hospitals, cosmetic, food industries, and pharmaceuticals units.
